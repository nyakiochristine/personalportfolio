Learning HTML: A Quick Start

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure for content on the web, using elements and tags.

Key Concepts

  • Tags: Enclose content to define elements (e.g., <p> for paragraphs, <h1> for headings).
  • Elements: Consist of a start tag, content, and an end tag.
  • Attributes: Provide additional information about elements (e.g., <img src="image.jpg" alt="My Image">).

Start by creating a basic HTML document:

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>My First Webpage</title>
                    </head>
                    <body>
                        <h1>Hello, World!</h1>
                        <p>This is my first webpage.</p>
                    </body>
                    </html>

Save this as `index.html` and open it in your browser to see the result.
